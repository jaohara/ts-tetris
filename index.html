Controls

It's Tetris! You know what's up.

  • Enter / N -> Start Game
  • Left / Right -> Move
  • Up -> Clockwise Rotate
  • E -> Counter-Clockwise Rotate
  • Down -> Soft Drop
  • Space -> Hard Drop
  • F -> Hold Piece / Swap Held Piece
  • Esc / P -> Pause

Debug Controls

  • PageUp -> Increase Level (adds 10 lines)
  • PageDown -> Decrease Level (removes 10 lines)
  • 0 -> Disable background
  • 9 -> Toggle between simple (default) and complex background (unoptimized)
  • 8 -> Display swatch of rendered minos
  • 7 -> Disable gravity
  • 6 -> Disable active piece glow
